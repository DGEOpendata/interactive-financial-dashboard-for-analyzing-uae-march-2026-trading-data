# Interactive Financial Dashboard for UAE March 2026 Trading Data

This project provides an interactive web-based dashboard for visualizing and analyzing the 'Trading Summary by Investor Type (March 2026, Daily Data)' dataset. The dashboard offers dynamic visualizations, advanced filtering options, and downloadable insights to cater to financial analysts, investors, regulatory bodies, and media professionals.

## Features
- Interactive data visualizations (line charts, bar charts, heatmaps) for analyzing trading activities.
- Advanced filtering options for customizing data analysis based on date ranges, investor types, and trading metrics.
- Downloadable reports and data exports in multiple formats.
- User-friendly interface for easy navigation and data exploration.

## Installation
1. Clone the repository:
   bash
   git clone https://github.com/your-username/uae-trading-dashboard.git
   cd uae-trading-dashboard
   
2. Install the required dependencies:
   bash
   pip install -r requirements.txt
   
3. Place the 'Trading_Summary_March_2026.xlsx' dataset in the project directory.

## Usage
1. Run the Flask application:
   bash
   python app.py
   
2. Open a web browser and navigate to `http://127.0.0.1:5000` to access the dashboard.
3. Use the filtering options to specify a date range and explore the data.

## Dataset
The dataset provides a detailed daily trading summary by investor type for the month of March 2026. It includes data on:
- Buy and sell values (in AED)
- Net values
- Buy and sell volumes
- Net volumes
- Number of buy and sell trades

## Dependencies
- Python 3.8+
- Flask
- Pandas
- Matplotlib
- Seaborn
